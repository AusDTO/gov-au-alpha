## About

### Tech

* Ruby 2.2.3
* Jekyll

## Setup

Ensure you have Git, Ruby

``` bash
git clone git@github.com:AusDTO/gov-au-alpha.git
cd gov-au-alpha
bundle install
```

## Running + developing

Serve app locally:

``` bash
jekyll serve
```

## Website
open http://localhost:4000/
